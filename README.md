# 🎵 Spotify Clone (HTML, CSS, JavaScript)

## 🚀 Overview
This is a frontend Spotify-inspired music player built using HTML, CSS, and JavaScript.  
It focuses on UI design and core JavaScript functionality such as playing music, switching songs, and controlling playback.

---

## 🛠️ Tech Stack
- HTML5  
- CSS3  
- JavaScript (ES6)  
- DOM Manipulation  
- HTML Audio API  

---

## ✨ Features
- ▶ Play / Pause music  
- ⏭ Next / Previous song control  
- 🎧 Playlist-based music switching  
- 🎨 Spotify-inspired dark UI  
- 📱 Responsive design for mobile and desktop  
- 🎵 Basic audio player functionality  

---

## 📂 Project Structure
/music        → Audio files
/images       → Song cover images
index.html    → Main HTML structure
style.css     → UI styling
script.js     → Player functionality

---

## 📚 What I Learned
- DOM manipulation using JavaScript  
- Working with HTML Audio API  
- Handling user events (click, play, pause)  
- UI layout design using CSS  
- Building interactive frontend component
